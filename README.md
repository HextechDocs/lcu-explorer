# LCU Explorer

---

A replacement for rift explorer which doesn't need restarting from the original contributors

## How to run
```
Make sure league of legends is open first then open lcu explorer.
```

## Build

```
yarn
```

then 
```
yarn build
```

## Run in dev mode
```
yarn 
```
then
```
yarn dev
```
